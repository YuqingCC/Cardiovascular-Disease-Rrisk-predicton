Disease Prediction using modified C4.5

***Important Notes****
The training set in our dataset is significantly large, for marker evaluation efficiency, we only run a small set of data.
See comments in code for more detail.

To compile and run our project:
1. install necessary package
   pip3 install numpy
   pip3 install pandas
   pip3 install sklearn.metrics
   pip3 install sklearn.model_selection
2. compile and run
   cd CSCI4144-ProjectCode-Group-12.zip
   python3 diseasePrediction.py
